# TK English Club Web

A private, static study player for AJ English lessons.

## Version 1 features

- Open an audio lesson directly from the device
- Open or paste a transcript
- Estimated sentence-by-sentence highlighting
- Manual sync correction buttons
- Search inside the transcript
- Playback-speed control
- Tap any English word for a simple dictionary definition
- Responsive layout for iPad, iPhone, and desktop
- Light/dark mode
- Audio and transcript stay on the user's device

## Files

Upload these files to the **root** of the GitHub repository:

- `index.html`
- `styles.css`
- `app.js`
- `README.md`

Do not upload only the ZIP file. GitHub hosting needs the extracted files.

## Important limitation

This first version estimates timing based on audio duration and sentence length. Exact automatic synchronization requires a timestamp file generated by speech-alignment software or a server-side transcription service. The built-in **Sync this sentence here** button lets you correct the timing while listening.
